# CSS Advanced

Ce projet est la continuité du projet HTML Advanced réalisé dans le cadre de la formation Holberton School.
L'objectif est de reproduire fidèlement une maquette Figma en utilisant uniquement le CSS pour mettre en forme la structure HTML créée précédemment.
Ce projet permet de pratiquer la mise en page, les couleurs, la typographie, Flexbox et les bonnes pratiques du développement front-end.
