# HTML Advanced - Tâche 0

## Présentation du projet

Ce projet constitue la première étape de l'apprentissage du HTML avancé dans le cadre de la formation Holberton School. L'objectif est de reproduire la structure d'une page web à partir d'une maquette Figma en utilisant uniquement du HTML sémantique, sans aucun style CSS. Cette approche permet de mieux comprendre l'organisation d'une page, l'utilisation des balises HTML5 et les bonnes pratiques en matière d'accessibilité.

## Objectifs

- Construire une page web uniquement avec du HTML.
- Respecter la structure de la maquette fournie sur Figma.
- Utiliser des balises HTML5 sémantiques.
- Préparer une base propre pour l'ajout du CSS dans les prochaines étapes.

## Technologies utilisées

- HTML5
- Git
- GitHub
- Figma
